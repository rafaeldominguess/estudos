# 🐍 Python Projects

Collection of Python projects demonstrating various programming concepts and problem-solving techniques.

## Projects

### 1. [Inventory Management System](./inventory-system/)

**Description:** A console-based inventory management system for tracking products, stock levels, and transactions.

**Features:**
- View all products in inventory
- Register product entries (purchases/restocking)
- Register product removals (sales)
- Input validation and error handling
- Real-time inventory updates

**Technologies:** Python 3.x

**Status:** ✅ Complete and Functional

---

## How to Run

1. Navigate to the project folder
2. Run: `python inventory-system/main.py`
3. Follow the on-screen menu
